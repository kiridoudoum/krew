# Mon IA Perso - Landing Page

A high-fidelity recreaction of a premium SaaS landing page, featuring a centered layout, floating UI mockup cards, and a sophisticated atmospheric background.

## Features
- **Strictly Centered Design**: Mathematical symmetry for a professional look.
- **Dynamic UI Mockups**: Asynchronous floating cards mimicking a real application.
- **Atmospheric Visuals**: Dotted grid background and soft radial glow animations.
- **Responsive Layout**: Optimized for both desktop and mobile viewing.

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- JavaScript

## Preview
Open `landing.html` in your browser to view the page.
